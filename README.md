# Portfolio Website (HTML, CSS, JavaScript)
## Overview
This is a personal portfolio website developed using HTML, CSS, and JavaScript. It
represents my personal profile as a BCA student, showcasing my skills, projects, and
professional links. The design is fully responsive and optimized for different devices,
providing smooth navigation and visually appealing layout.
## Features
- Responsive navigation bar with dropdown menu for mobile view
- Animated hero sectionwith introduction and social media icons 
-  Downloadable CV button for resume access
-  About section describing educational background and goals
-  Skills section highlighting core technical abilities
-  Projects section linking to GitHub repositories with live previews
-  Contact section with direct email link and social handles
## File Structure
| File/Folder | Description | | index.html | Main HTML structure of
the website | | style.css | Contains styling rules and layout design | | script.js | Handles
animations, mobile menu toggle, and interactivity | | images/ | Stores profile picture and
related visuals | | files/ | Contains downloadable CV (PDF format) |
## How It Works
1. The navigation bar links to sections within the page for easy access.
2. The hero section introduces the user with a short bio and image.
3.  The CV button allows visitors to download the resume directly.
4.  Each project is displayed with title, description, and GitHub link.
5.   The contact section provides email and social media links for outreach.
## Technologies Used
- HTML5 for structure
- CSS3 for layout, animations, and responsiveness
- JavaScript for dynamic elements and interactivity
- Font Awesome for social icons
## How to Run
1. Download or clone the repository from GitHub.
2.  Ensure all folders and files (HTML,CSS, JS, images, and CV) are in the same directory.
3.  Open the `index.html` file in any web browser.
4.   Explore different sections through the navigation bar.
